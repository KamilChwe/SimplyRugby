Main Screen

1. Details Fields

This section allows you to add and edit Player and Coach data.
Be sure to fill out ALL of the fields and then press Add Player or Add Coach.
If any fields (either coach or player) are empty then you'll get an error.

2. Display List + Buttons

This allows you to see each Player and Coach in the Database, use the buttons at the bottom (Display Players and Display Coaches) to see the entries
You can select any player or coach and edit or delete their details.
To edit details please selected a player or coach in the list and use the fields on the left in section 1 and to submit press Edit Selected.
To delete an entry please select a player or coach from the list and press Delete Selected.

3. Notifying Coaches

This allows you to send emails to the coaches as to notify them of any important information.
Enter your message in the box, there is no character limits.
You can either send it to all Emails in the database or selected few.

4. Side Buttons

Logout allows you to return to the Login screen, you'll recieve a pop up asking if you're sure since any unsaved data will be lost.
