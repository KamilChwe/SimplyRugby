Coach Screen Documentation

Login Screen

1. Login and Password

To enter the Coach Screen in the app you will have to input your Coach Login and Password provided by the Admin.
Upon entering the correct Login and Password you will be moved to the main Coach Screen.
If the creditentials are incorrect then a warning will pop up telling you that either your login OR password was incorrect

Main Screen

1. Player List

This section allows you to select a player and see/edit their Skill Data.
Upon selecting a player you will see the numbers on the right change. Coaches cannot edit Name, Date of Birth, or Squads! If you need to change any details of a player please contact an Admin.

2. Rating Fields

As mentioned before, when you select a player all their scores and comments will appear in the fields.
You can alter any rating and the comment for any player in the database.
There is also a "Last Changed" text box to show when the information you are viewing was changed, this applies to both Admin and Coach changes!
Do not leave any fields empty as this will cause in an error.

3. Filter Options

The filters allow you to display only the squad you are interested in, simply click on the Dropbox and select which squad it is that you want to view.
All the players from that squad will appear in the Display List above.
By default the list displays all the players, when you pick a filter and then want to see all Players again press "Show All".

4. Side Buttons

Logout allows you to return to the Login screen, you'll recieve a pop up asking if you're sure since any unsaved data will be lost.
The Help button takes you to this page.
